mac下基础环境安装与使用(CPU)

目前飞桨支持的环境(2023.04)

  • macOS 版本 10.11/10.12/10.13/10.14 (64 bit) (不支持 GPU 版本)
  • Python 版本 3.7/3.8/3.9/3.10 (64 bit)
  • pip 或 pip3 版本 20.2.2 或更高版本 (64 bit)
